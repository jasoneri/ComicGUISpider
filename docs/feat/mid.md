## 使用

### 1. 创建 / 切换工作流

右上角下拉框输入名称创建新工作流，或选择已有工作流切换  
点击 💾 保存按钮持久化，🗑️ 删除按钮移除当前工作流

### 2. 添加规则

从右侧**规则库**按 Lane 分类浏览可用规则，点 `+` 添加到画布对应 Lane

- 点击画布上的规则按钮 → 右侧切换为**详情面板**，可查看/修改参数
- 详情面板中的 ↑↓ 徽标可调整同一 Lane 内的执行顺序，✕ 移除规则
- 点击画布空白处返回规则库

### 3. 启用自动化

拨动开关后出现提示：

- **保存后生效** — 下次进入对应阶段时自动执行已配置的规则
- **立即激活** — 点击提示中的 ▶ 按钮，从「选择网站」阶段开始运行

::: tip 运行中画布会高亮当前所在 Lane，
Lane 旁 ▶ 按钮可手动触发该 Lane 的规则执行
:::

## Lane 与 Timeline

画布底部的节点对应下载流程的五个决策阶段：

| Lane | 阶段 | Timeline 值 |
|------|------|------------|
| A | 选择网站 | 90 |
| B | 输入搜索 | 100 |
| C | 选择书本 | 210 |
| D | 选择章节 | 310 |
| E | 后处理 | 600 |

规则挂载到对应 Lane 后，工作流运行至该阶段时自动按优先级依次执行

## 内置规则

| 标签 | 名称 | Lane | 说明 |
|------|------|------|------|
| C1 | Select First Book | C | 自动选择第一本 |
| C2 | Select First Book2 | C | 同上（备用） |
| D1 | Select Latest n Episode | D | 选择最新 n 个章节，参数 `num` 可调 |
| E1 | To CBZ | E | 下载完成后转换为 CBZ 格式 |

::: details 工作流状态机
```
IDLE ──▶ RUNNING ──▶ WAITING（等待 Lane 执行）
  ▲                      │
  │        ◀─────────────┘
  │         规则执行完毕
  └──── FINISHED 重置
```
:::
