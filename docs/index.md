## 功能说明

- 简易配置就能使用
- 开预览后随便点点就能下载，还能基于预览窗口进网站看
- 通过加减号，`0 全选`，`-3 选倒数三个` 等输入规则，能方便指定选择
- 基于翻页保留，翻页就像已塞进了购物车一样
- 虽然任务是顺序流，但内置重启很方便，加上多开更方便
- 读剪贴板方式流，字如其名
- 去重，加标识符等
